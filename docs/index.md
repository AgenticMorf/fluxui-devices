# Flux UI Devices

FluxUI front-end for managing devices and sessions using diego-ninja/laravel-devices. Designed for the Laravel Livewire Starter Kit with Flux UI components.

## Features

- Device management: view all devices that have accessed the account
- Session management: view and manage active browser sessions
- Remote sign out: sign out of specific devices or all other devices
- Location display: session location when available
- FluxUI components with Livewire Volt
- Custom device type icons (no Flux Pro required)
